### NLP Inference Service (`/backend-services/nlp-inference-service`)

**Environment Variables**

To run the NLP service securely, you must configure the following environment variable:

-   `ALLOWED_ORIGINS`: A comma-separated list of URLs that are permitted to make requests to the API. This is a critical security measure to prevent Cross-Origin Resource Sharing (CORS) attacks.

    -   **Format**: `URL1,URL2,URL3`
    -   **Example for Local Development**:
        ```bash
        ALLOWED_ORIGINS="http://localhost:3000,http://127.0.0.1:3000"
        ```
    -   **Example for Production**:
        ```bash
        ALLOWED_ORIGINS="https://your-frontend-domain.com,https://www.your-frontend-domain.com"
        ```

**Note**: If this variable is not set, the API will not accept requests from any origin, effectively disabling it.
